### How to use ###

json-server REST api가 필요합니다. 
```
npm intall -g json-server (글로벌로 설치해줍니다.)
```
json서버 포트는 3001로 사용해줍니다.
```
json-server --watch ./src/db/data.json --port 3001
```
npm은 어떤 포트를 사용해도 무방합니다.
